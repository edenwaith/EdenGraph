EdenGraph Help

Graph 2D Mathematical Expressions

EdenGraph can graph equations in 2D space, which are displayed on an X-Y Cartesian plane. You only need to type in the right hand side of the equation. To graph a sine wave, type sin(x) and click on the Graph button.

EdenGraph does not currently graph polar coordinates or 3D plots. Refer to Functions and Constants to learn about what tools are available to plot 2D graphs.
